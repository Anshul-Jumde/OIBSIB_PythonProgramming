# Advanced Smart Weather App

## Description

Advanced Smart Weather App is a Python-based GUI application developed using CustomTkinter. It provides real-time weather information for any city using an online weather API. The application displays temperature, humidity, wind speed, weather conditions, feels-like temperature, weather tips, and dynamic themes based on weather conditions.

## Features

* Modern GUI using CustomTkinter
* Search weather by city name
* Real-time weather information
* Temperature display with color indicators
* Humidity display
* Wind speed display
* Weather condition display
* Weather emojis
* Feels Like temperature
* Weather tips based on conditions
* Last searched city display
* Reset button
* Input validation
* Dynamic day/night theme
* Weather-based background colors
* Dynamic application title

## Technologies Used

* Python
* CustomTkinter
* Requests Library
* Datetime Module
* wttr.in Weather API

## How to Run

1. Install Python

2. Install required libraries:

py -m pip install customtkinter requests

3. Run the application:

py weather_app.py

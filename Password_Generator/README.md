# Password Generator

## Description

Password Generator is a Python-based GUI application developed using CustomTkinter. It generates secure and random passwords based on user-selected criteria such as password length, uppercase letters, lowercase letters, numbers, and symbols.

## Features

* User-friendly GUI using CustomTkinter
* Custom password length selection
* Uppercase letters option
* Lowercase letters option
* Numbers option
* Symbols option
* Random password generation
* Password strength indicator
* Copy password to clipboard
* Clean and modern interface

## Technologies Used

* Python
* CustomTkinter
* Random Module
* String Module
